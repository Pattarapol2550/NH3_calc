สูตรที่ใช้คำนวณ — Formula Reference

▾

① Compressor Power (3-phase, ค่าคงที่ V=385, PF=0.86)

P_comp (kW) = 1.732 × 385 × I × 0.86 ÷ 1000 // = I × 0.5728 (สูตรลัด)

② Pressure Conversion

P_abs (Pa) = (P_gauge × 98066.5) + 101325 // 1 kg/cm² = 98066.5 Pa, Patm = 101325 Pa

③ CoolProp Property Lookup (NH₃)

from CoolProp.CoolProp import PropsSI T_sat = PropsSI("T","P",P_low,"Q",1,"Ammonia") − 273.15 h1 = PropsSI("H","P",P_low, "T",T_suction+273.15,"Ammonia") / 1000 h2 = PropsSI("H","P",P_high,"T",T_discharge+273.15,"Ammonia") / 1000 h3 = PropsSI("H","P",P_high,"T",T_liquid+273.15,"Ammonia") / 1000 h4 = h3 # isenthalpic expansion

④ Cycle Performance

COP = (h1 − h4) / (h2 − h1) Q_e = P_comp × COP [kW] Q_H = Q_e + P_comp [kW] ṁ = Q_e / (h1 − h4) [kg/s]

⑤ Notes

• V = 385 V, PF = 0.86 เป็นค่าคงที่ของระบบ • สูตรลัด: P_comp (kW) = I × 0.5728 • CoolProp ใช้ Helmholtz energy EOS — แม่นเทียบ REFPROP • h₂ ใช้ DT จริง → COP = actual COP ของเครื่อง • SH < 0 → liquid slugging → ตรวจ EXV ด่วน • SC < 0 → flash gas ก่อน EXV → ประสิทธิภาพลด
