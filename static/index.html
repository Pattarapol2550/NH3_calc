สูตร & Logic การ assume

▾

① P_comp (V=385, PF=0.86 คงที่)

P_comp = 1.732 × 385 × I × 0.86 ÷ 1000 = I × 0.5728 [kW]

② ST — กรอกหรือ assume SH = 5K

ถ้ากรอก ST: SH = ST − T_sat(SP) h1 = PropsSI("H","P",P_low,"T",ST+273,"Ammonia") ถ้าเว้นว่าง: SH = 5 K // assume T1 = T_sat(SP) + 5 h1 = PropsSI("H","P",P_low,"T",T1,"Ammonia")

③ DT — กรอกหรือ assume η_is = 0.70

ถ้ากรอก DT: h2 = PropsSI("H","P",P_high,"T",DT+273,"Ammonia") η_is = (h2s−h1)/(h2−h1) // คำนวณย้อนกลับ ถ้าเว้นว่าง: h2s = PropsSI("H","P",P_high,"S",s1,"Ammonia") h2 = h1 + (h2s−h1) / 0.70 // assume η=0.70

④ Liquid Temp — กรอกหรือ assume SC = 0

ถ้ากรอก T_liq: SC = T_cond − T_liq h3 = PropsSI("H","P",P_high,"T",T_liq+273,"Ammonia") ถ้าเว้นว่าง: SC = 0 // liquid = T_cond h3 = hf(P_high) // Q_e จะ conservative (ต่ำกว่าจริง)

⑤ COP & Output

COP = (h1−h4) / (h2−h1) Q_e = P_comp × COP [kW] ṁ = Q_e / (h1−h4) [kg/s]
